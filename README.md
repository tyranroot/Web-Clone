## рҹ“Ң **Overview**

**Web-Clone** is a professional-grade website cloning tool that allows you to download entire websites for offline viewing, backup, or security research purposes. It features a modern GUI, multi-threaded downloading, and intelligent link parsing.

> вҡ пёҸ **IMPORTANT**: This tool is for **EDUCATIONAL PURPOSES** only. Only clone websites you own or have explicit permission to copy.

---

## вңЁ **Features**

| Feature | Description |
|:-------:|:------------|
| рҹҢҗ **Full Site Cloning** | Download complete websites including HTML, CSS, JS, images |
| рҹ“Ҡ **Crawl Depth Control** | Set how deep to crawl (1-5 levels) |
| вҡЎ **Multi-Threaded** | Fast parallel downloading |
| рҹ“Ғ **Custom Output** | Choose where to save cloned files |
| рҹҺЁ **Modern GUI** | Professional dark theme interface |
| рҹ“ҹ **Live Console** | Real-time download progress and logs |
| рҹӣЎпёҸ **Domain Lock** | Only downloads same-domain content |
